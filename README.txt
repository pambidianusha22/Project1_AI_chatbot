🤖 Rule-Based AI Chatbot

📌 Project Overview

The Rule-Based AI Chatbot is a beginner-level Artificial Intelligence project developed using Python and Jupyter Notebook. The chatbot interacts with users by responding to predefined text inputs using rule-based logic and dictionary-based response handling.

This project demonstrates the basic concepts of conversational AI, logical decision-making, loops, and user interaction handling.

---

🎯 Objective

The objective of this project is to create a simple chatbot capable of:

- Responding to predefined user inputs
- Handling greetings and exit commands
- Running continuously using loops
- Demonstrating basic AI interaction flow

---

🛠 Technologies Used

- Python
- Jupyter Notebook

---

⚙ Working Principle

The chatbot works using predefined rules stored inside a Python dictionary. Each user input is matched with a corresponding chatbot response using dictionary-based lookup.

The chatbot:

1. Accepts user input
2. Processes the input
3. Searches for matching responses
4. Displays the appropriate reply
5. Continues interaction until the exit command is entered
